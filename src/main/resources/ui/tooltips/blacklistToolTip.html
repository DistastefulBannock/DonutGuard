The blacklist is an easy-to-use setting that blocks out
specific classes and resources from reaching the mutators.

The blacklist in DonutGuard uses regular expressions so
the developer has more power and control.

The strings added to this list are compared to the paths
of different entries in the jar. This means that it uses
forward slashes(/) rather than dots(.) and includes file
extensions.

Clicking this button will open a regex helper site for
your use.
